Overview

Usage

Reference

lang Module

bind

Signature: * ⇒ ∅

Binds the specified value (auto-quoted) to an existing symbol .

call

Signature: (*) ⇒ *?

Calls operator defined in scope (*).

debug

Signature: ∅ ⇒ ∅

Toggles debug mode.

debug?

Signature: ∅ ⇒ B

Returns true if debug mode is on, false otherwise.

define

Signature: * ⇒ ∅

Defines a new symbol , containing the specified value (auto-quoted).

delete

Signature: ⇒ ∅

Deletes the specified symbol .

eval

Signature: S ⇒ *?

Parses and interprets S.

exit

Signature: ∅ ⇒ ∅

Exits the program or shell.

from-json

Signature: S ⇒ *

Converts a JSON string into MiNiM data.

import

Signature: ⇒ ∅

Imports the a previously-loaded module , defining all its symbols in the current scope.

load

Signature: S ⇒ *?

Parses and interprets the specified MiNiM file S, adding .min if not specified.

module

Signature: (*) ⇒ ∅

Creates a new module based on quotation (*).

scope

Signature: (*) ⇒ (*)

Unquotes (*) creating a new scope and pushes a new copy of (*) on the stack.

scope?

Signature: (*) ⇒ (B)

Returnes true if (*) defines a scope, false otherwise.

sigils

Signature: ∅ ⇒ (S+)

Returns a list of all sigils defined in the ROOT scope.

source

Signature: ⇒ (*)

Display the source code of symbol (if it has been implemented a MiNiM quotation).

symbols

Signature: ∅ ⇒ (S+)

Returns a list of all symbols defined in the ROOT scope.

to-json

Signature: (*) ⇒ S

Converts a quotation into a JSON string S.

with

Signature: (2) (1) ⇒ *?

Applies quotation (2) within the scope of (1).
